robokudo.annotators
===================

.. py:module:: robokudo.annotators

.. autoapi-nested-parse::

   RoboKudo annotators package.

   This package provides annotators for processing and analyzing sensor data.
   It supports:

   * Core annotator functionality
   * Visualization and display
   * Data processing and filtering
   * Object detection and tracking
   * Spatial analysis and mapping

   The package contains:

   * Core annotators for basic functionality
   * Visualization annotators for display
   * Processing annotators for data analysis
   * Testing annotators for debugging
   * Utility annotators for common tasks
